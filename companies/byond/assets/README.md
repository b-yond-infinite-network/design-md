# B-Yond assets

Drop the following SVGs here (names referenced by `DESIGN.md` / `DESIGN.dark.md`):

- `logo.svg` — full-color logo for light backgrounds.
- `logo-dark.svg` — inverted logo for dark backgrounds.
- `wordmark.svg` — type-only mark.
- `favicon.svg` — simplified glyph.

Prefer SVG. Only commit raster (PNG/JPG) when SVG is not viable (e.g., `social-card.png`).
